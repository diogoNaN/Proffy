## Tecnologies

>NodeJS + TypeScript
>* express
>* knex
>* cors
>* sqlite3



## Start

```sh
# Install all dependencies:
$ yarn

# Start server:
$ yarn start

# Run Migrations:
$ yarn knex:migrate

# Rollback of Migrations:
$ yarn knex:migrate:rollback
```
